Projects

  • Logic Wealth

    pralfredo/LogicWealth

    Developed a constraint-rich portfolio optimizer that constructs equity portfolios under realistic investment mandates, including sector exposure limits, ESG/liquidity filters, beta ranges, volatility ceilings, cardinality requirements, and weight bounds. Modeled portfolio selection using Boolean decision variables and arithmetic constraints, framing portfolio construction as an SMT-style feasibility and optimization problem. Built a 50-asset large-cap universe and interactive dashboard that outputs selected holdings, allocation weights, share counts, risk metrics, sector exposure, and exclusion explanations. Implemented static GitHub Pages demo mode and local FastAPI solver mode, separating portfolio presentation from backend solver execution.

    Python Z3 FastAPI JavaScript Logic & Quant

    pralfredo/logic-visualizer

    Developed an interactive web-based logic visualization platform for constructing and evaluating propositional formulas and logical circuits in real time. Designed dynamic visual representations of logical operators and truth-functional relationships, enabling intuitive exploration of formal reasoning processes. Implemented clientside parsing, state management, and live circuit evaluation to provide immediate feedback as users modified logical expressions. Engineered an educational interface that bridges symbolic logic and computational visualization, making abstract logical structures accessible through interactive graphics.

    Logic Epistemology HTML CSS Javascript

Publication

Dynamic Pricing Algorithms for Online Set Cover

arXiv · 2409.15094 · 2024

We consider dynamic pricing algorithms as applied to the online set cover problem. In the dynamic pricing framework, we assume the standard client server model with the additional constraint that the server can only place prices over the resources they maintain, rather than authoritatively assign them. In response, incoming clients choose the resource which minimizes their disutility when taking into account these additional prices. Our main contributions are the categorization of online algorithms which can be mimicked via dynamic pricing algorithms and the identification of a strongly competitive deterministic algorithm with respect to the frequency parameter of the online set cover input.
